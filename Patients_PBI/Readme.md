## Project Overview
The **Patient Data Explorer Dashboard** is an interactive Power BI project that visualizes healthcare patient data.  
It provides insights into patient demographics, visit history, medical conditions, and billing, helping explore healthcare metrics interactively.

This is a **beginner/fresher Power BI project** to practice data modeling, DAX, and dashboard design with a healthcare sample dataset.

---

## Key Features
- **Patient Analysis:** ID, gender, age, weight, BMI  
- **Medical Insights:** Primary condition, chronic conditions, preventive care  
- **Visit Analysis:** Last visit date, annual visits, days since last visit  
- **Billing & Insurance:** Average bill, total billing, insurance type  
- **Interactive Visuals:** Filters, slicers, and responsive KPIs
