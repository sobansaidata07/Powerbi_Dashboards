# Employee Analytics Dashboard

This repository contains a **very basic Employee Analytics Dashboard** built to analyze workforce data for a small organization of approximately **27 employees**. The dashboard provides a quick snapshot of employee distribution, salaries, promotions, and designations.

---

## Dashboard Overview

The dashboard focuses on simple, high-level HR insights suitable for quick reviews by HR or management teams.
