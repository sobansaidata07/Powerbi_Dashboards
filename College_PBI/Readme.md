# Multi-Institute Student Analytics Dashboard

This repository contains a Business Intelligence (BI) dashboard designed to analyze student data across multiple educational institutes in Singapore. The dashboard provides insights into enrollment, demographics, qualifications, expenses, and time-based trends to support data-driven decision-making.

---

## Dashboard Overview

The dashboard aggregates student data from major Singapore institutes:

- Nanyang Technological University (NTU)
- National University of Singapore (NUS)
- Singapore Management University (SMU)
- Singapore University of Technology and Design (SUTD)

### Key Metrics
- **Total Students:** 1,000  
- **Total Expenses:** 457K  
- **Gender Distribution:** ~50.2% Male, ~49.8% Female  

Interactive visualizations and slicers allow users to explore data by institute, qualification, gender, status, reason, and time period.

## Usage Instructions
1. Download the Power Bi File.
2. Open the dashboard file (`.pbix`) in Power BI Desktop  
3. Refresh the data connections  
4. Use slicers to filter by institute, gender, qualification, or time period  

---

## Use Case

This dashboard is intended for academic administrators and analysts to monitor enrollment patterns, demographic distributions, and expense trends across multiple institutes in Singapore.
